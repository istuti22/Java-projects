A simple Number Guessing Game where the player tries to guess a randomly generated number within a certain range. 
The game provides hints like “Too High” or “Too Low” until the correct number is guessed. 

#Features 
Random number generation within a configurable range 
User input validation Feedback after each guess 
Tracks number of attempts Option to play again 

#How to Play 
The computer randomly selects a number between a given range (default: 1–100). 
You try to guess the number by entering your guess. 
The program will tell you if your guess is: 
1. Too High 
2. Too Low
3. Correct 
Keep guessing until you find the correct number. After guessing correctly, you’ll see how many attempts it took.
