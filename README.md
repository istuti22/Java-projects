#  Student Record Management System (Core Java)

A simple console-based Java project to manage student records.

## Features
•⁠  ⁠Add, display, search, and sort student records  
•⁠  ⁠Uses arrays, loops, and Bubble Sort algorithm  
•⁠  ⁠Demonstrates core Java concepts like classes and objects  

## How to Run
1.⁠ ⁠Compile: ⁠ javac StudentManagement.java ⁠
2.⁠ ⁠Run: ⁠ java StudentManagement ⁠
