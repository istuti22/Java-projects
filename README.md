A simple Core Java console-based project that allows users to add, view, search, and cancel hotel reservations.
It demonstrates the use of OOP concepts and Collections Framework (HashMap).

 Features:

1.Add new reservations
2.View all bookings
3.Search by customer name
4.Cancel reservations
5.Prevent duplicate room bookings

 Concepts Used:

1.Classes & Objects
2.HashMap for storing reservations
3.Switch-case and loops for menu handling
4.Encapsulation for data security

 How to Run

1.⁠ ⁠Open the project in any Java IDE or terminal.


2.⁠ ⁠Compile and run:

javac HotelReservation.java
java HotelReservation
